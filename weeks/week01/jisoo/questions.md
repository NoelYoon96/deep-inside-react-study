### 2. 컴포넌트화는 어느 기준까지 진행하는 것이 효율적일지?
#### 현재 실무에서 사용중인 방법

1. Layout 컴포넌트를 사용하며, 모든 스타일은 인라인 형식으로 작성
container/wrapper는 className 으로 부여한다.


- 1단계 : 디자인 토큰 단위로 typo, color 정의
- 2단계 : Layout 컴포넌트 (styled components + styled system)
  ex. Column, Grid, CenterRow 등 정의
- 3단계 : Button, Input, Tag, Radio 등
- 4단계 : 반복되는 역할을 가진 컴포넌트의 경우에만 컴포넌트화
  - 해당 기능 폴더에 넣기 / 다른 곳에서 하나라도 사용하는 경우에는 common 폴더로 이동
